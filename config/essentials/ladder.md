# Essentials ladder (Ponytail-inspired)

Inspired by [DietrichGebert/ponytail](https://github.com/DietrichGebert/ponytail). Do not vendor.

Before building, ask in order:

1. Does this need exist?
2. Can we reuse existing block or native feature?
3. Can stdlib / platform / official docs solve it?
4. Can an existing dependency solve it?
5. Can one line or config change solve it?
6. What is the minimum that works?

Never cut: security, validation, error handling, accessibility, VERIFY blocks.
