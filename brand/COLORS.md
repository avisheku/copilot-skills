# Skillrig brand colors

| Token | Hex | Use |
|-------|-----|-----|
| `ink` | `#0A1628` | Primary dark / logo field |
| `sea` | `#0E4D5C` | Structure / secondary |
| `signal` | `#3DDC97` | Live / pass / accent |
| `ember` | `#F4A261` | Upgrade / attention |
| `fog` | `#E6EEF2` | Light surfaces |
| `mist` | `#8BA3B5` | Muted labels |

CSS variables:

```css
:root {
  --sr-ink: #0A1628;
  --sr-sea: #0E4D5C;
  --sr-signal: #3DDC97;
  --sr-ember: #F4A261;
  --sr-fog: #E6EEF2;
  --sr-mist: #8BA3B5;
}
```
