# SkillsForge brand colors

| Token | Hex | Use |
|-------|-----|-----|
| `skills` | `#1B3A6B` | "Skills" wordmark / blue rings |
| `forge` | `#3D3D3D` | "Forge" / hammer / charcoal |
| `deep` | `#0F1724` | Dark banner fields |
| `fog` | `#F4F6F8` | Light surfaces |
| `mist` | `#6B7280` | Tagline / muted |

CSS variables:

```css
:root {
  --sf-skills: #1B3A6B;
  --sf-forge: #3D3D3D;
  --sf-deep: #0F1724;
  --sf-fog: #F4F6F8;
  --sf-mist: #6B7280;
}
```
